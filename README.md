## Setup Environments (Python 3.10)

1. Install python prerequisites
```
$ pip install -r requirements.txt
```

2. Run 3D Viewer
```
(venv)$ cd python
(venv)$ python viewer3d.py --config viewer3d.cfg
```

3. Test URDF Parser
```
(venv)$ cd python/tests
(venv)$ python test_urdf.py
```
