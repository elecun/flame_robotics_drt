## Setup Environments (Python 3.10)

1. Install python prerequisites
```
$ pip install -r requirements.txt
```

2. Launch 3D Viewer
```
$ python -m venv venv
$ source ./venv/bin/activate
(venv)$ cd python
(venv)$ python viewer3d.py
```

3. Launch Simulation Manager
```
(venv)$ python manager.py
```
